## About
Just another social sharing plugin.


## Add a brand

 - Install https://github.com/s-a/scream-js
 - Add a file to ```/ico``` folder
 - Add the file to ```/ico/.scream.js``` folder
 - execute ```scream .scream.js --build```
 - execute ```node bundle.js```
 - execute ```grunt build```
